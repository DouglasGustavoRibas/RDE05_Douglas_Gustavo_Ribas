Imagens e Listas

Imagem é uma representação visual de algo, ja a imagem digital é a representação de uma imagem bidimencional usando números binários codificados de modo a permitir seu armazenamento, transferência, impressão ou reprodução, e seu processamento por meios eletrônicos.

As imagems podem possuir diferentes formatos:

  • .BMP
  • .TIFF
  • .GIF
  • .SVG
  • .ICO
  • .JPEG
  • .PNG

Marcação de Imagens

As tags de imagens e seus parâmetros definem as imagens e suas propriedades ex: < img src="imagem.extensão" width = "largura" height = "altura" alt="descrição" >

As tags figure e figcaption foram definidas a partir do HTML 5 ex:

<figure>

<img src="imagem.extensão" width = "largura em pixel" height="altura em pixel" alt="descrição">

<figcaption>Legenda</figcaption>

</figure>

Marcação de favicon

Define o icone que aparece na barra de título ex:

<link real="icon" href="caminho para o icone"> type = "image/x-icon"

Marcação de Listas

As tags de listas podem definir:

  • Listas não-ordenadas
  • listas ordenadas
  • lista detalhada

Listas não-ordenadas

<ul>

<li>Item 1</li>

<li>Item 2</li>

</ul>

Listas ordenadas

<ol type="elemento">

<li>primeiro</li>

<li>segundo</li>

</ol>

Listas detahada

<dl>

<dt>item 1</dt>

<dd>Detalhamento do item 1</dd>

</dl>
